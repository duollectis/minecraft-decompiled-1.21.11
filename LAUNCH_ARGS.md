# Аргументы запуска клиента Minecraft 1.21.11

Полный список аргументов командной строки, принимаемых [`Main.java`](src/main/java/net/minecraft/client/main/Main.java).

---

## Обязательные аргументы

| Аргумент | Тип | Описание |
|---|---|---|
| `--accessToken <token>` | `String` | Токен авторизации сессии. Можно передать `0` для офлайн-режима |
| `--version <version>` | `String` | Строка версии игры, например `1.21.11` |

---

## Директории

| Аргумент | Тип | По умолчанию | Описание |
|---|---|---|---|
| `--gameDir <path>` | `File` | `.` (текущая директория) | Рабочая директория игры — сохранения, настройки, логи, моды |
| `--assetsDir <path>` | `File` | `<gameDir>/assets/` | Путь к папке с ассетами (текстуры, звуки, переводы). Без неё игра запустится с розовыми текстурами |
| `--resourcePackDir <path>` | `File` | `<gameDir>/resourcepacks/` | Путь к папке с ресурспаками |
| `--assetIndex <id>` | `String` | — | Имя файла индекса ассетов из `assets/indexes/` (без `.json`). Для 1.21.11 это `29`. Значение указано в поле `assetIndex.id` в JSON-манифесте версии |

---

## Игрок и сессия

| Аргумент | Тип | По умолчанию | Описание |
|---|---|---|---|
| `--username <name>` | `String` | `Player<timestamp>` | Игровой никнейм |
| `--uuid <uuid>` | `String` | UUID из никнейма (офлайн) | UUID игрока без дефисов. Если не указан или невалиден — генерируется из никнейма |
| `--xuid <xuid>` | `String` | `""` | Xbox User ID (для Microsoft-аккаунтов) |
| `--clientId <id>` | `String` | `""` | Идентификатор клиента для телеметрии |

---

## Окно

| Аргумент | Тип | По умолчанию | Описание |
|---|---|---|---|
| `--width <px>` | `Integer` | `854` | Ширина окна в пикселях |
| `--height <px>` | `Integer` | `480` | Высота окна в пикселях |
| `--fullscreenWidth <px>` | `Integer` | — | Ширина в полноэкранном режиме |
| `--fullscreenHeight <px>` | `Integer` | — | Высота в полноэкранном режиме |

---

## Флаги (без значения)

| Аргумент | Описание |
|---|---|
| `--fullscreen` | Запустить в полноэкранном режиме |
| `--demo` | Запустить в демо-режиме (ограниченное время игры) |
| `--disableMultiplayer` | Отключить вкладку мультиплеера |
| `--disableChat` | Отключить чат |
| `--checkGlErrors` | Включить проверку ошибок OpenGL |
| `--offlineDeveloperMode` | Режим офлайн-разработчика (без авторизации) |
| `--renderDebugLabels` | Отображать отладочные метки рендера |
| `--jfrProfile` | Включить профилирование через Java Flight Recorder |
| `--tracy` | Включить профилировщик Tracy |
| `--tracyNoImages` | Отключить захват изображений в Tracy |

## QuickPlay — быстрый вход в мир/сервер

Можно указать **только один** из трёх вариантов одновременно.

| Аргумент | Тип | Описание |
|---|---|---|
| `--quickPlayPath <path>` | `String` | Путь к файлу QuickPlay (JSON с параметрами быстрого входа) |
| `--quickPlaySingleplayer [worldId]` | `String?` | Быстрый вход в одиночный мир по его имени |
| `--quickPlayMultiplayer <host:port>` | `String` | Быстрое подключение к серверу мультиплеера |
| `--quickPlayRealms <realmId>` | `String` | Быстрое подключение к Realms-серверу по ID |
